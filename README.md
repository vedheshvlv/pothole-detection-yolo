# 🚧 Pothole Detection using DIP and YOLO

## 📌 Overview
This project detects potholes using:
- Digital Image Processing (DIP)
- YOLO (Deep Learning)

## ⚙️ Features
- Pothole detection from road images
- Severity classification (Low / Medium / High)
- Performance evaluation using Precision, Recall, Accuracy

## 🧠 Technologies Used
- Python
- OpenCV
- NumPy
- Ultralytics YOLO

## 📊 Results
| Model | Precision | Recall | Accuracy |
|------|----------|--------|----------|
| DIP | Low | High | Misleading |
| YOLO | Improved | Balanced | Better |

## 🚀 How to Run

### Train YOLO - python train_yolo.py

### Detection - python yolo_detection.py

### Evaluation - python yolo_evaluation.py

## 📁 Dataset
Dataset used:
RDD / Road Damage Dataset

(Note: Dataset not included in repo due to size)
